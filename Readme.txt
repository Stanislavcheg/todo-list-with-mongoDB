Hello, it a simple "toDo lists" application.

For the client side were used: HTML, CSS, Bootstrap, jQuery, RequireJS, Mustache.js, Bootbox.js

For the server side were used: Node.js, Express.js, MongoDB, Mongoose
 	
To run at localhost:
1. Install node js.
2. Access server folder through the command prompt.
3. Enter 'node server' to run localhost server.
4. Open any browser and enter 'http://localhost:80/' in the address line.
